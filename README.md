# Akij Takaful Life Insurance PLG

---

## Installation

1. Clone repository

```bash
https://github.com/WorkFortBD/akij-takaful-frontend.git
```

2. Go to the folder

```bash
cd akij-takaful-frontend
```

3. Install Node packages

```bash
npm i
```

4. Run Local server with development change watching...

```bash
npm run dev
```

5. Open browser

```bash
http://localhost:3000
```
